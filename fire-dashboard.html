Fire Spread Animation Dashboard

A client-facing web application for visualizing interactive fire spread simulations with secure authentication.

Project Demo

The dashboard allows the user to select an ignition point and weather scenario (dropdown hidden due to recording) in order to view the modeled fire for those conditions.

Project Overview

Developed a comprehensive fire spread animation dashboard for Filsinger Energy Partners to share simulation results with clients. The application provides an intuitive interface for visualizing complex fire behavior data with optimized performance and secure access controls.
